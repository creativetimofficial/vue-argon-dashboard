# Charts

Chart components are wrappers over Vue Chart.js library and enhanced visually to look inline with the dashboard style.

Initialization:

// Charts
import {
  salesChart,
  ordersChart
} from "@/components/Charts/Chart";

export default {
  data() {
    return {
      salesChartID: "salesChart",
      ordersChartID: "ordersChart"
    };
  },
}
